Map:

The map shows the latest statistics for the selected municipality. All the latest data is based on the Finnish Statistics Center from the year 2022 except employment rate, which is from the year 2021.

  • Crime rate takes all reported crimes from year 2022 into account from the selected municipality.
  • Employment rate is calculated with only employed and unemployed personnel in the selected municipality. It does not consider the whole population.

Chart:

The chart shows selected data in a chart. All data is presented from the years 2000 - 2022, except employment rate which is only to the year 2021.

  • The default value for the chart is Finland population data.
  • The chart has two presentation styles which are dot/line chart and bar chart. Selection can be made from the top menu.
  • Presented data can be selected from the top menu buttons.
  • The chart can be exported as a svg file using the "Export chart" button from the top menu.

Municipality:

Data in the chart can be specified to use any municipality in Finland.

  • The municipality of the data to be presented in the chart can be selected from "Municipality" button from the top menu.
  • If nothing is selected, there will be used default value "Finland".
  • Municipality is chosen by typing letters to the search field and the application will suggest municipalities based on the input.
  • Selected municipality will be presented in the button itself.
